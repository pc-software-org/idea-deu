# Changelog

Der Abschnitt der jeweils gebauten `plugin_version` (aus `config/product.json`)
wird zum „What's new"-Text auf dem JetBrains Marketplace. Nur `## <version>`-
Überschriften und `-`-Aufzählungspunkte werden unterstützt.

## 2026.2.0.1.1

- Übersetzungen an das umstrukturierte Modul-Layout von 2026.2 angepasst
